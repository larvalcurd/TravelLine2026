# Fighters

Fighters — это консольная игра, в которой игрок создает бойцов из доступных рас, классов, оружия и брони, а затем запускает между ними автоматическую битву.

## Как запустить

```bash
dotnet run --project Fighters
```

## Команды

| Команда | Описание |
| --- | --- |
| `add` | Создать и добавить бойца |
| `fight` | Начать битву между добавленными бойцами |
| `list` | Показать список добавленных бойцов |
| `help` | Показать доступные команды |
| `exit` | Выйти из игры |

### Основные файлы

| Файл | Назначение |
| --- | --- |
| `Program.cs` | Хранит список добавленных бойцов, считывает команды и вызывает нужные сервисы |
| `CharacterCreator.cs` | Запрашивает у игрока параметры бойца и создает объект `Fighter` |
| `Battle.cs` | Запускает цикл боя, броски инициативы, атаки, вывод статуса и определение победителя |
| `GameData.cs` | Хранит доступные расы, классы, оружие и броню |
| `Fighter.cs` | Содержит характеристики бойца, логику атаки, логику получения урона и состояние жив/мертв |
| `AttackReport.cs` | Описывает результат одной атаки |

## Ход игры

1. Игра запускается и выводит список доступных команд.
2. Игрок использует `add` для создания бойцов.
3. Во время создания бойца игрок выбирает:
   - имя
   - расу
   - класс
   - оружие
   - броню
4. Игрок может использовать `list`, чтобы посмотреть добавленных бойцов.
5. Игрок использует `fight`, чтобы начать бой.
6. Бой продолжается до тех пор, пока в живых не останется только один боец.
7. Последний выживший боец объявляется победителем.

## Характеристики бойца

Боец состоит из четырех элементов:

```text
боец = раса + класс + оружие + броня
```

Итоговые характеристики рассчитываются следующим образом:

| Характеристика | Формула |
| --- | --- |
| Макс. здоровье | `race.Health + fighterClass.Health` |
| Урон | `race.Damage + fighterClass.Damage + weapon.Damage` |
| Броня | `race.Armor + fighterClass.Armor + armor.Armor` |
| Бонус инициативы | `race.Initiative + fighterClass.Initiative` |

## Правила боя

Каждый бой делится на раунды.

В начале каждого раунда:

1. Каждый живой боец бросает инициативу.
2. Инициатива рассчитывается как `d20 + InitiativeBonus`.
3. Бойцы действуют в порядке убывания инициативы (от наибольшей к наименьшей).

В ход бойца:

1. Выбирается случайный живой противник.
2. Боец атакует цель.
3. Урон от атаки умножается на случайное значение от `0.8` до `1.1`.
4. Вероятность критической атаки составляет `20%`.
5. При критической атаке используется множитель урона `2.0`.
6. Броня цели снижает получаемый урон.
7. Урон не может быть меньше `0`.

Урон, получаемый целью:

```text
фактическийУрон = max(входящийУрон - броняЦели, 0)
```

Если здоровье бойца падает до `0`, он выбывает из боя.
